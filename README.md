# D3_Angular_template
This is a starting point for D3 + Ng...

Standard init:
1. Go to API => npm install --save
1b. node server.js //(nodemon server.js for convenience of development ;)

2. Go to Frontend => npm install --save
2b. ng serve 

Enjoy! :)
